## 安装：
1. 确保已安装 [PopClip](https://www.popclip.app/)
2. 鼠标双击 BaiduTranslate.popclipext
3. 点击 [安装“百度翻译”]
4. 前往 [百度翻译开放平台](https://api.fanyi.baidu.com/doc/21) 获取 APP ID 和密钥
5. 在插件选项中填入 APP ID 和密钥，以及配置译文显示方式和替换方式
